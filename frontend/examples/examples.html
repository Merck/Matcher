Example Queries

This page shows examples of what kinds of searches are possible.

Example 2: Run search

Example 3: Run search

Example 4: Run search

Example 5: Run search
Find all pairs where hydrogen changes to anything, on the 4-position of a pyridyl substructure.




Example 6: Run search
Find all pairs where hydrogen changes to anything, on an aromatic carbon, where both molecules in the pair have a carboxylic acid anywhere in the molecule.




Example 7: Run search
Find all pairs that vary at the amide. Require <= 8 non-H atoms in the variable (green-highlighted) fragment. Group transforms by transformed fragment.




Example 8: Run search
Find all pairs where a 1-naphthyl changes to any substructure containing a 6-membered aromatic ring attached to the core via a tertiary amide.
